## III. POWER AUTOMATE WORKFLOWS

**File:** `email_automation/power_automate_flows/FLOW_CONFIGURATION_MASTER.md`

### Flows Delivered:

#### Flow 1: Email Categorization & Routing
- Auto-detect email category (Court, Creditor, Heir, Professional, Debt Collector)
- Apply Outlook categories and priority flags
- Save to SharePoint Communication Log
- Trigger template selection flow

#### Flow 2: Template Selection & Auto-Fill
- Retrieve case data from SharePoint
- Select appropriate template based on category
- Auto-fill variables (case number, names, dates, amounts)
- AI Builder data extraction (amounts, dates, names)
- Create draft email for user review
- Teams notification

#### Flow 3: Creditor Claim Tracker
- Parse creditor information from emails
- Create/update SharePoint Creditor Claims list
- Set 30-day response reminders
- Track claim status

#### Flow 4: Court Deadline Tracker
- Daily recurrence (8:00 AM)
- Check upcoming hearings (7-day window)
- Send reminders (7-day, 1-day, day-of)
- Teams urgent notifications

#### Flow 5: Invoice Approval Workflow
- Detect invoice attachments
- Extract invoice data (AI Builder)
- Save to SharePoint Invoices library
- Create approval task
- Generate payment confirmation draft

#### Flow 6: Heir Update Scheduler
- Monthly recurrence (1st of month, 9:00 AM)
- Generate status updates from SharePoint data
- Create personalized drafts for each heir
- Notify administrator to review

---
