## AUTOMATION NOTES FOR OUTLOOK/POWER AUTOMATE

**Workflow Setup:**

1. **Auto-Categorize Professional Emails:**
   - Detect emails from known professional contacts (attorneys, CPAs, appraisers)
   - Auto-tag as "Probate - Professionals"
   - Track by professional type (legal, accounting, appraisal, real estate)

2. **Template Selection Logic:**
   - New contact → Template 1 (Initial inquiry)
   - Engagement → Template 2 (Agreement)
   - Need documents → Template 3 (Document request)
   - Check status → Template 4 (Status update)
   - Invoice received → Template 5 (Invoice review)
   - Problem → Template 6 (Dispute)
   - Project complete → Template 7 (Thank you)
   - Need referral → Template 8 (Referral request)
   - Court hearing → Template 9 (Coordination)
   - End relationship → Template 10 (Termination)

3. **Auto-Fill Variables:**
   - Professional name and firm
   - Estate information (case number, value, decedent name)
   - Contact information
   - Current date
   - Invoice details (from PDF parsing)

4. **Invoice Management:**
   - Auto-detect PDF invoices in emails
   - Extract invoice number, date, amount
   - Create approval workflow
   - Track payments in SharePoint
   - Generate payment confirmation emails

5. **Professional Database:**
   - Name and firm
   - Service type (attorney, CPA, appraiser, realtor, etc.)
   - Contact information
   - Hourly rate / fee structure
   - Services provided
   - Invoices sent/paid
   - Performance notes

**Power Automate Flow:**

```
TRIGGER: Email from professional contact
↓
ACTION: Categorize by professional type
↓
CONDITION: Contains invoice attachment?
   YES → Extract invoice data, create approval task
   NO → Analyze for response need
↓
ACTION: Suggest appropriate template
↓
ACTION: Auto-fill known variables
↓
ACTION: Present to user for customization
↓
ACTION: On approval, send and log
↓
ACTION: Set follow-up reminder if needed
```

**Tracking and Reporting:**
- Total fees paid by professional type
- Outstanding invoices
- Service completion status
- Court appearance calendar
- Professional contact directory
